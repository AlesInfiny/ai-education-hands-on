## リポジトリのフォルダー構成

```text
root
│  README.md ................................. このファイル
│
├─design-specification-templates ............ COBOL コードをモダナイズするための設計書テンプレートを配置したフォルダー
│
├─guides

│  │
│  └─images
│
└─legacy-codes .............................. GitHub Copilot を使ってモダナイズする COBOL コードのフォルダー
```
