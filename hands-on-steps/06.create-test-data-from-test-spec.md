# 設計書とテスト仕様書からテストデータを作成する

1. GitHub Copilot Chat を開きます。
1. GitHub Copilot に次の質問をします。

    ```plaintext
    #file:design-doc-main.md および #file:test-specification-main.md をインプットとし、 #file:test-data-template.md に基づいて、テストデータを作成してください。 #file:test-specification-main.md に定義されたすべてのテストケースについて、テストデータを作成してください。データセットは可能な限り使いまわしてください。また、 #file:test-data-template.md のコメント行で囲まれた内容は入力例です。入力例を参考にして出力し、出力したものから入力例は削除してください。
    ```

    ※コピー＆ペーストでファイルがうまく参照されない場合は、プロンプトに `#` を直接入力してファイルを選択してください。

1. GitHub Copilot によって作成された回答の右上の `...` をクリックし、「新しいファイルに挿入する」を選択します。
1. `testdata-main.md` というファイル名を入力し、保存します。

    ※新しいファイルに挿入したとき、回答内容が最後まで反映されないことがあります。その場合は回答をコピー＆ペーストしてください。
